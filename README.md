This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, `npm i`.

Then, create an `.env` file with the following variables:

```
NEXT_PUBLIC_RUTTER_CLIENT_ID=
NEXT_PUBLIC_RUTTER_CLIENT_SECRET=
```

Finally, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000/login](http://localhost:3000/login) with your browser to see the result.
